# checkpoint_management_simple_cluster

This resource allows you to execute Check Point Simple Cluster.

## Example Usage


```hcl
resource "checkpoint_management_simple_cluster" "example" {
  name = "cluster1"
  color = "yellow"
  version = "R80.30"
  os_name = "Gaia"
  cluster_mode = "cluster-xl-ha"
  firewall = true
  ipv4_address = "17.23.5.1"
}
```
